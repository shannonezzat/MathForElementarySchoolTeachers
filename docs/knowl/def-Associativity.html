Definition 2.6.6.

We say an operation \(\square\) is associative exactly when it's the case that when \((a \square b) \square c =d\) then \(a (\square b \square c) =d\text{;}\) that is, \((a \square b) \square c = a a\square (b \square c)\) (grouping performing the operation doesn't matter).
