Let \(n\) be even. Then \(n=2k\) for some whole number \(k\text{.}\) We want to show that \(n^2 = 2(\text{integer})\) which would show that it is always even when \(n\) is even. Consider

\begin{equation*} n^2 = (2k)^2 = 4k^2 = 2(2k^2). \end{equation*}

Since \(2k^2\) is a whole number, we have written \(n^2\) as an even whole number.
