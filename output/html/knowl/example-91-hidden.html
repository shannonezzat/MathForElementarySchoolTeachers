Let's say we want to add the fractions \(\frac{3}{4}\) and \(\frac{5}{6}\text{.}\) First, we need to find equivalent fractions with a common denominator, which in this case is the least common denominator (LCD) of 4 and 6. The LCD is 12, so we will find equivalent fractions with a denominator of 12:

\begin{equation*} \frac{3 \times 3}{4 \times 3} = \frac{9}{12}, \quad \frac{5 \times 2}{6 \times 2} = \frac{10}{12} \end{equation*}

Now that we have equivalent fractions with the same denominator, we can add them just as we did in the previous example:

\begin{equation*} \frac{9}{12} + \frac{10}{12} = \frac{9 + 10}{12} = \frac{19}{12} \end{equation*}

Similarly, if we want to subtract the fractions, we can do so by subtracting the numerators while keeping the same denominator:

\begin{equation*} \frac{9}{12} - \frac{10}{12} = \frac{9 - 10}{12} = \frac{-1}{12} \end{equation*}
