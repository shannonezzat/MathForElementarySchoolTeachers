In math, we're more concerned with if-then statements where both the if-part and then-part are open sentences with variables. For example "if \(n\) is an odd integer then \(n^2\) is also odd." In these cases, there's an assumed universal quantifier before these: "for all integers \(n\) if \(n\) is odd then \(n^2\) is odd." We don't usually write the quantifier, but it's there in the background. Since it's a universially quantified statement, we want it to be true for all inputs into our variables.
