Solution 4.3.NaN.1.

First, set up the cross multiplication equation:

\begin{equation*} 3(15) = 5x \end{equation*}

Now, solve for \(x\text{:}\)

Multiply \(3\) by \(15\text{:}\) \(3(15) = 45\text{.}\)

So, the equation becomes: \(45 = 5x\text{.}\)

Divide both sides by \(5\text{:}\) \(45/5 = 9\text{.}\)

So, \(x = 9\text{.}\)

Therefore, the value of \(x\) in the proportion is \(9\text{.}\)
