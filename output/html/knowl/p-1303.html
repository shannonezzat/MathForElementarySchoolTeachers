And here are examples of how to write a terminating decimal as a fraction:

  1. 0.5 = 5/10 = 1/2: Here, we express 0.5 as the fraction 5/10, which simplifies to 1/2.

  2. 0.75 = 75/100 = 3/4: We can express 0.75 as the fraction 75/100, which simplifies to 3/4.

  3. 0.875 = 875/1000 = 7/8: The decimal 0.875 can be expressed as the fraction 875/1000, which simplifies to 7/8.

  4. 0.2 = 2/10 = 1/5: The decimal 0.2 can be expressed as the fraction 2/10, which simplifies to 1/5.
