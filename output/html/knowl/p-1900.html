Remember that if we have a conditional statement "if P then Q" it is logically equivalent to its contrapositiveL "if not Q then not P". Thus we can express a biconditional as "If P then Q *and* if not P then not Q". We can rewite the result in the exercise above to be the following theorem that will be very useful for us:

Theorem 7.2.13.

Let \(m\) be a natural number. If \(m\) is an \(n\)th power then each exponent in its prime factorization has a factor of \(n\text{.}\) If \(m\) is not an \(n\)th power, then there is a prime number in its prime factorization whith an exponent with no factor of \(n\text{.}\)
