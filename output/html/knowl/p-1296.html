Suppose we have a number \(a\) and we are indeed curious about what \(a\) raised to a negative exponent would mean. We want to keep the same rules that we've used before, where if you have the same base and different exponents, you add them. This would imply that \(a^1 \cdot a^{-1}\) should equal \(a^{1+(-1)}\text{,}\) which is \(a^0\text{.}\) And anything raised to the power of zero is one. So what could we multiply \(a\) by to get one? The answer is \(\frac{1}{a}\text{.}\) Hence, whenever we see a negative exponent like this, it implies the reciprocal of the base raised to the corresponding positive exponent. In general, \(a^{-n} = \frac{1}{a^n}\text{.}\)
