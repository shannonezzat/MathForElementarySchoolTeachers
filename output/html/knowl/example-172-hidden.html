1. Even if we know there are two solutions to the equation \(x^2=4\text{,}\) we say that \(\sqrt{4}=2\text{.}\)

  2. We say that the two solutions to the equation \(x^2=14\) are \(x=\sqrt{14}\) and \(x=-\sqrt{14}\) since the principal root is always positive. Note that since \(14\) isn't a square number we cannot write this as an integer. Later, we will learn about simplifying roots.

  3. We know that \(x^3=8\) has one solution since the exponent 3 is odd, so we say that \(\sqrt[3]{8}=2\text{.}\) Similarly we say that \(\sqrt[3]{-8}=-2\text{.}\)
