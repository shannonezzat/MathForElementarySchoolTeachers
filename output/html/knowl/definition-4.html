Definition 1.1.9.

If \(P\) is a statement, then the negation of \(P\text{,}\) written \(\sim P\) is a statment with the opposite truth value of \(P\) (but with the same context). If \(P\) is an open sentence, then \(\sim P\) is an open sentence with the opposite truth value for each variable input.
