Item 2.

Let \(n\) be odd. Then \(n=2k+1\) for some whole number \(k\text{.}\) We want to show that \(n^2 = 2(\text{integer})+1\) which would show that it is always odd when \(n\) is odd. Consider

\begin{equation*} n^2 = (2k+1)^2 = (2k+1)(2k+1) = 4k^2+4k+1 = 2(2k^2+2k)+1. \end{equation*}

Since \(2k^2+2k\) is a whole number, we have written \(n^2\) as an odd whole number.
