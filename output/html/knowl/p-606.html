Now, let's "break" this chocolate bar between the 3rd and 4th vertical block thus giving us 3 blocks in the first section and 5 in the second (of course, we still have the same amount of chocolate). So we can view this now as \(6 \times (3+5)=48\text{.}\) But we could calculate the number of blocks separately in each of the two sections, and then add them together. Obviously we would still have the same amount of chocolate. So there are \(6 \times 3+ 6 \times 5 = 18 + 30 = 48\) blocks of chocolate.
