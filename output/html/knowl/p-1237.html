A proportion is an equation that states that two ratios are equal. For example, if we know that the ratio of apples to oranges in a fruit basket is 3:2 and there are 12 apples, we can set up a proportion to find the number of oranges. We set up the proportion as follows: 3/2 = 12/x. Solving for x (which represents the number of oranges), we find that there are 8 oranges.
