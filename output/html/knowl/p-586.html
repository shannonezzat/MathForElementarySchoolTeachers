Now we have 3 squares, and 2 ones. Can we write this as \(32_{five}\text{?}\) No! Be careful here. We are missing a place; we have no fives, so we need to indicate that with the digit 0: so we should write \(302_{five}\text{.}\) Note that \(32_{five}\) is 3 fives and 2 ones, which is 17. This is why that 0 digit is important!
